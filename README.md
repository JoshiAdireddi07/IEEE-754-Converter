# 🧮 IEEE 754 Converter 

An interactive web application that converts decimal numbers to IEEE 754 floating-point binary representation and vice versa.

## ✨ Features
- Real-time conversion (Decimal ↔ IEEE 754)
- Supports 32-bit and 64-bit precision
- Color-coded breakdown: Sign, Exponent, Mantissa
- Interactive UI with animations
- Works entirely client-side (no backend)

---

## 🧠 How It Works
1. Enter a decimal or binary value.
2. JavaScript processes conversion using `ArrayBuffer` and `DataView`.
3. The result displays instantly with visual highlights.

---

## 🧰 Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** None (fully client-side)  
- **Tools:** VS Code, GitHub Pages
